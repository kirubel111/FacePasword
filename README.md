
‎I wrote a script that takes conventional passwords which are easy to break.
My script uses facial features and mixes them with the password to make it a strong and unbreakable password.


# Installation Step :-

## if Windows 💠 :
Open Command Prompt
```bash
python -m venv .venv
.venv\Scripts\activate.bat
```

## if Linux 🐧:-

### For Ubuntu/Debian:
```bash
sudo apt update
sudo apt install python3-venv
```
### For Fedora/CentOS:
```bash
sudo dnf install python3
```
# Create the Virtual Environment:-
```bash
python3 -m venv .venv
source .venv/bin/activate
```
# Install the Library:
```bash
pip install cmake
pip install dlib
pip install face_recognition
pip install opencv-python numpy
```
# Run code
```bash
python3 face.py
```
